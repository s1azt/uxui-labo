# 📝 お知らせ・コラムの更新方法（簡易版）

## 🎯 概要
お知らせとコラムの内容は、**外部ファイル（`/data/news.js` と `/data/columns.js`）** で一元管理されています。  
これらのファイルを編集するだけで、**トップページ、お知らせ一覧、コラム一覧すべてに自動反映**されます。

---

## 📰 お知らせの更新方法

### 対象ファイル
```
/data/news.js
```

### 更新手順

1. **`/data/news.js`** を開く
2. 配列の**一番上**に新しいお知らせを追加
3. 以下の形式をコピー＆ペーストして編集:

```javascript
{
    date: "2026.02",                    // 日付
    status: "実施予定",                  // ステータス
    statusColor: "orange",              // 色（orange/green/blue/purple/indigo）
    category: "アンケート",              // カテゴリー
    title: "新しいお知らせのタイトル",
    description: "お知らせの説明文をここに記載します。",
    link: null                          // リンク先（なければ null）
},
```

4. ファイルを保存
5. **ブラウザをハードリロード**（Ctrl + Shift + R / Cmd + Shift + R）

### 反映先
- ✅ トップページの「最新情報」セクション（最新3件）
- ✅ お知らせ一覧ページ（全件表示）

---

## ✍️ コラムの更新方法

### 対象ファイル
```
/data/columns.js
```

### 更新手順

1. **`/data/columns.js`** を開く
2. 配列の**一番上**に新しいコラムを追加
3. 以下の形式をコピー＆ペーストして編集:

```javascript
{
    date: "2026.02",
    link: "./column4.html",                        // コラムページのパス
    image: "https://images.unsplash.com/...",      // サムネイル画像URL
    authorInitial: "山",                           // 著者イニシャル
    authorName: "山田 太郎",                        // 著者名
    authorColor: "from-orange-500 to-red-500",    // アイコン色
    title: "新しいコラムのタイトル",
    description: "コラムの説明文をここに記載します。",
    featured: false,                               // 注目記事（true/false）
    accentColor: "orange"                          // アクセントカラー
},
```

4. ファイルを保存
5. **ブラウザをハードリロード**（Ctrl + Shift + R / Cmd + Shift + R）

### アクセントカラーの選択肢
- `"purple"` : 紫系
- `"blue"` : 青系
- `"green"` : 緑系
- `"orange"` : オレンジ系
- `"pink"` : ピンク系
- `"indigo"` : インディゴ系

### 反映先
- ✅ トップページの「UXコラム」セクション（最新3件）
- ✅ コラム一覧ページ（全件表示）

---

## ⚠️ 重要な注意点

### 1. キャッシュ対策
データファイルを更新したら、**必ずブラウザをハードリロード**してください:
- **Windows/Linux**: `Ctrl + Shift + R`
- **Mac**: `Cmd + Shift + R`

システムは自動的にキャッシュを回避する仕組みになっていますが、初回表示時は手動リロードが必要です。

### 2. 古いデータの扱い
- 古いお知らせやコラムは**削除せず残してください**
- アーカイブページで全件表示されます
- 新しいものほど**上に追加**することで、自動的に時系列順に表示されます

### 3. JavaScriptの文法エラーに注意
- 最後のカンマ（`,`）を忘れずに
- 引用符（`"`）を正しく閉じる
- 配列の閉じ括弧（`]`）とセミコロン（`;`）を確認

---

## 🔧 トラブルシューティング

### Q. 更新したのに反映されない
**A.** ブラウザのハードリロード（Ctrl + Shift + R）を試してください。

### Q. ページが真っ白になった
**A.** JavaScriptの文法エラーの可能性があります。ブラウザの開発者ツール（F12）でエラーを確認してください。

### Q. 画像が表示されない
**A.** 画像URLが正しいか確認してください。Unsplashなどの外部サービスのURLを使用できます。
